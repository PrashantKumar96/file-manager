File Upload System

Features:

  • Has a Profile Page showing no. of files uploaded.
  • Has a feature of deleting uploaded file.
  • Login/Logout and session time limit of 30 minutes.
  • Uploaded data of one user is not visible to other.
  • Restriction of uploading some specific file types.
  • Restriction on file size. Files larger than 20 MB are not allowed.
